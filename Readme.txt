Проект: https://github.com/SerjikS/Countries
К сожалению, я не могу на месте у вас протестировать работоспособность на вашем сервере.

Необходимо включить в компонентах службы IIS/ASP.
Необходимо в Диспетчере служб IIS создать сайт. 
В папку с сайтом разархивировать содержимое Countries\Countries_WebServer\Countries_WebServer. 
Рекомендуемый порт: 8081
Тип входа: ClearText
В пуле приложений в дополнительных параметрах выбрать Удостоверение: LocalSystem
База уже присутствует в папках, клиент и сервер сами всё подключат, ничего кроме запуска сервера делать не требуется.
На всякий случай бэкап базы прилагается в корне. Де-факто это пустая база с названием Countries.mdf (без таблиц и данных).
Программа сама заполнит базу если она пустая.
После запуска клиента в документах создаётся файл Link.txt. В него будет записывать адрес для подключения. Совершить перезапись адреса сайта для подключения можно прямо из программы.
